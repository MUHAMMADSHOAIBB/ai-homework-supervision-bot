## 1、产品背景

 AI专注力学习机器人，以番茄工作法为核心框架，以儿童专注力培养理论为指导，通过视觉感知、多模态交互和智能引导机制，实现对学习过程的实时监督、动态反馈和长期习惯养成。核心功能是监督孩子写作业和学习，解决的问题：

1. 家长无法长期陪伴监督，难以及时发现孩子学习过程中的分心行为；
2. 孩子自我管理能力较弱，容易出现发呆、走神、玩手机、频繁离座等情况；
3. 传统番茄钟工具仅提供时间管理功能，缺乏对学习状态的实时感知与动态引导能力；
4. 家长只能看到学习结果，无法了解学习过程中的专注度变化和行为表现。

​    最终产品希望做成一个“孩子端 + 家长端”配合使用的 AI 学习陪伴系统。孩子端主要有两种形态：一种是放在桌面上的硬件学习 bot，通过摄像头、麦克风、扬声器和表情屏陪伴孩子学习；另一种是轻量版的小程序或 App，可以直接利用手机、平板的摄像头完成番茄钟、学习状态识别和 AI 提醒。孩子学习时，系统会按照番茄钟流程管理学习节奏，识别孩子是否在座位、是否写作业、是否走神、是否玩手机、是否趴桌疲劳，并通过语音、表情或虚拟形象进行提醒和鼓励。家长端则通过小程序或 App 查看学习面板，包括当前是否在专注学习、已经学习多久、今天完成了几轮番茄钟、分心了几次，以及一段时间内的专注力变化情况。

## 2、软件模块

| 功能               | 定义                                                         | 备注 |
| :----------------- | :----------------------------------------------------------- | :--- |
| 实时状态识别       | 连续识别、输出“人在位/离座、看桌面/走神、写字活动、坐姿、手机、疲劳风险”等状态 |      |
| 番茄钟流程         | 预备、专注、休息、恢复、完成                                 |      |
| 规则引擎           | 用事件与阈值触发对应的action                                 |      |
| 话术生成           | 规则给出意图，智能体生成最终话术                             |      |
| 语音播报           | 把话术转为音频                                               |      |
| 面部表情状态机     | 根据当前的action 和 状态机的状态，产生表情                   |      |
| 家长端展示         | 查看专注时长、打断事件、姿势趋势、阶段总结                   |      |
| 数据存储与隐私保护 | 本地、最小化存储，使用前家长授权、记录历史可删除等           |      |

## 3、流程

实时识别 → 番茄钟状态机 → 规则引擎 → 话术生成 → TTS → TFT 表情反馈 → 家长端展示

### 番茄钟状态机

| 主状态       | 进入条件            | 动作                             | 退出条件                 |
| :----------- | :------------------ | :------------------------------- | :----------------------- |
| IDLE         | 用户未开始          | 显示待机表情，等待开始           | 点击开始或语音唤醒       |
| PREPARE      | 开始学习前 1–2 分钟 | 引导坐好、准备文具、确认桌面 ROI | 孩子准备好或准备时间到   |
| FOCUS        | 进入专注阶段        | 计时、识别、轻量反馈             | 达到专注时长或中断严重   |
| BREAK        | 正常休息            | 放松话术、休息表情、关闭干预识别 | `休息时长` 到时          |
| RESUME_CHECK | 休息后收口          | 检查是否回到书桌                 | 回到桌前则再次进入 FOCUS |
| COMPLETE     | 本轮/本次结束       | 总结语、家长端摘要               | 会话结束                 |
